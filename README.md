# 🌿 MoodGarden – UI/UX Mobile App Design

MoodGarden is a mobile application designed to help users track their moods, reflect through journaling, and improve emotional well-being through a calm and engaging digital experience.

---

## 🎨 Figma Prototype (Interactive Demo)
👉 https://www.figma.com/proto/onzjiQf7DcNlYKSrI7RWiZ/MoodGarden-Mobile-App-UI-UX

---

## 📱 Key Features

- 🌸 Mood tracking with emotion selection
- 📖 Journaling with text and image support
- 🗓 Calendar-based mood history tracking
- 🧩 Predefined templates (Gratitude, Self-care, Food diary)
- 🎯 Clean and user-friendly UI design
- 🔄 Interactive prototype simulating real app flow

---

## 🧠 Design Focus

- User-centered design
- Emotional engagement
- Simple and intuitive navigation
- Consistent UI components
- Accessibility and readability

---

## 🛠 Tools Used

- Figma (UI/UX Design & Prototyping)
